### 1. Позиционирование и Геометрия (Где блок находится?)

Сначала пишем свойства, которые определяют место блока в макете и его размеры. Это позволяет сразу понять «каркас».

- display, position, flex, grid
- width, height, max-width, min-width
- margin, padding
- z-index


### 2. Типографика (Как выглядит текст?)

Затем описываем всё, что касается букв внутри блока.

- font-family, font-size, line-height, font-weight
- text-align, text-transform
- color, letter-spacing

### 3. Оформление (Визуальные эффекты)

В самом конце — «украшательства», которые не влияют на размеры блоков.

- background, border, border-radius
- box-shadow, opacity
- transition, animation
